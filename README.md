# Micro-Interactions Lab: Advanced UI Components 🧪

Welcome to my UI Research & Development space. This repository is a dedicated playground built to explore the boundaries of modern front-end styling using **React**, **TypeScript**, and **Tailwind CSS**.

The core focus of this project is to master the small details — **micro-interactions, dynamic states, and animations** — that separate generic interfaces from production-grade, premium user experiences.

## 🎯 Key Engineering Goals

Unlike standard tutorial projects, this lab was created with specific architectural challenges in mind:

1. **Advanced Tailwind Ecosystem:** Harnessing the power of Tailwind CSS v4, including arbitrary values, state modifiers (`focus-within`, `invalid`), and complex peer-to-peer element relations (`peer-*`).
2. **Vanilla-First Animations:** Creating high-end, smooth user experiences using native CSS transitions and hardware-accelerated animations instead of bloating the bundle with heavy external libraries.
3. **State & Logic Separation:** Ensuring that components like dynamic inputs or interactive cards maintain a strict separation between UI presentation and clean React business logic.

---

## 🛠️ The Tech Stack

- **Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS v4 + PostCSS
- **Type Safety:** TypeScript (for predictable component props and strict variants)
- **Deployment:** GitHub Actions (Automated CI/CD pipeline triggering builds on every push)

## 📦 Featured Components

### 1. The "Sentient" Input (Inteligentne Pole Tekstowe)

_Status: In Development ⚙️_
A highly interactive input field designed for forms where UX matters.

- **Floating Label:** Smooth CSS-driven label transition based on input focus and placeholder states (`peer-placeholder-shown`).
- **Glow & Shadow Effects:** Dynamic, hardware-accelerated focal rings that adapt to the field's validity.
- **Micro-interactivity:** Built-in password strength/validation transitions reacting live to user input.
